# Smart WiFi Manager（智能wifi管理）

一个基于Electron开发的Windows WiFi管理桌面应用程序，提供便捷的WiFi连接和管理功能。

## 功能特性

### 🎯 精简版核心功能
- ⭐ **智能收藏**: 将常用WiFi添加到收藏栏，一键快速连接
- 📂 **保存管理**: 显示和管理所有已保存的WiFi网络
- 🔄 **快速刷新**: 一键刷新网络列表和状态
- ⌨️ **全局快捷键**: 使用 `Alt+W` 快速唤起应用
- 💡 **状态识别**: 高亮显示当前连接的网络
- 🔒 **安全连接**: 支持加密WiFi密码输入连接
- ✨ **简洁界面**: 极简设计，专注核心功能
- ⚡ **快速响应**: 优化连接判断速度，2-5秒内完成

## 系统要求

- Windows 10 或更高版本
- 管理员权限（用于WiFi操作）

## 安装说明

### 方式一：使用预编译版本
1. 下载最新的安装包
2. 运行安装程序并按照提示完成安装
3. 启动应用时选择"以管理员身份运行"

### 方式二：从源码构建
1. 确保已安装 Node.js (版本 16 或更高)
2. 克隆或下载项目源码
3. 在项目目录下运行以下命令：

```bash
# 安装依赖
npm install

# 开发模式运行
npm start

# 构建Windows安装包
npm run build:win

# 构建便携版
npm run build:portable
```

## 使用指南

### 基本操作

1. **启动应用**: 双击桌面图标或从开始菜单启动
2. **扫描WiFi**: 应用会自动扫描可用的WiFi网络
3. **连接WiFi**: 
   - 点击WiFi列表中的网络进行连接
   - 对于加密网络，输入密码后点击连接
   - 对于已保存的网络，直接点击即可连接
4. **断开连接**: 点击顶部的断开按钮或右键当前连接的WiFi

### 收藏功能

1. **添加收藏**: 点击WiFi列表中的⭐按钮将网络添加到收藏栏
2. **快速连接**: 点击收藏栏中的WiFi卡片快速连接
3. **移除收藏**: 点击收藏卡片右上角的×号移除收藏

### 快捷键

- `Alt+W`: 显示/隐藏应用窗口
- `F5`: 刷新WiFi列表
- `ESC`: 关闭对话框

### 系统托盘

应用支持最小化到系统托盘：
- 双击托盘图标显示主窗口
- 右键托盘图标显示菜单选项

## 权限说明

本应用需要管理员权限来执行以下操作：
- 获取已保存的WiFi配置
- 连接和断开WiFi
- 管理WiFi配置文件
- 删除已保存的网络

**✅ 完全无需位置权限**：应用使用PowerShell Get-NetConnectionProfile和WMI查询技术，能够准确获取当前WiFi连接状态，完全不依赖需要位置权限的Windows API，在关闭位置服务的情况下仍能完整功能运行。

这些权限仅用于WiFi管理功能，不会收集或传输任何个人数据。

## 故障排除

### 常见问题

1. **无法获取已保存的WiFi列表**
   - 确保以管理员身份运行应用
   - 检查WiFi适配器是否正常工作
   - 尝试刷新WiFi列表

2. **连接失败**
   - 确认网络在范围内且密码正确
   - 尝试删除已保存的网络配置后重新连接
   - 检查网络是否可用

3. **快捷键不工作**
   - 检查是否有其他应用占用了相同快捷键
   - 尝试重启应用

### 日志查看

应用运行时的错误信息会显示在控制台中。如需查看详细日志：
1. 按 `F12` 打开开发者工具
2. 查看 Console 标签页中的错误信息

## 技术架构

### 🏗️ 精简架构设计
- **框架**: Electron（轻量化配置）
- **前端**: 
  - HTML5（50行精简结构）
  - CSS3（300行现代样式）
  - JavaScript（200行核心逻辑）
- **WiFi管理**: 
  - Windows netsh（配置管理、连接操作）
  - PowerShell Get-NetConnectionProfile（当前连接检测）
  - 简化网络验证（ping连通性测试）
- **特色优势**:
  - ✅ 完全无需位置权限
  - ✅ 快速连接判断（2-5秒）
  - ✅ 极简代码结构（易维护）
  - ✅ 专注核心功能（高效率）
